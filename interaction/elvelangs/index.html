Topic Source

This project is one of our tangible interaction project and we cooperate with a organ donation organization of Oslo named Stiftelsen Organdonasjon, theme to introduce the idea of dead organ donation to more people. And this Interactive device finally showed in the light show called Elvelangs and attracted a lot of people.

One For Seven

After their introduction, we thinked various of ideas. Finally, we are attracted to the idea one for seven which means if one people die and donate his body, that will save seven lifes. Basic on that, the fund we have and our technical limitations, we got our final idea.

Hug

Well, this is our final idea: hug the organ donation, save seven lifes, we are really into this interactive behavior HUG, it’s warm, inspiring and really matches the theme. This is the final video on the show.

Arduino Code
<!DOCTYPE arduino>
int pinButton1 = 2;
int pinButton2 = 3;
int pinButton3 = 4;

int pinLed1 = 9;
int intro = true;
int outro = false;
int value = 0;
void setup() {
  // put your setup code here, to run once:;
  pinMode(pinButton1, INPUT);
  pinMode(pinButton2, INPUT);
  pinMode(pinButton3, INPUT);


  pinMode(pinLed1,OUTPUT);
  Serial.begin(9600);

}

void loop() {
  // put your main code here, to run repeatedly:
  int countHugA = 0;
  if(digitalRead(pinButton1)) countHugA+=1;
  if(digitalRead(pinButton2)) countHugA+=1;
  if(digitalRead(pinButton3)) countHugA+=1;

  bool hugA= countHugA>1;
  Serial.println(countHugA);
  if (hugA){
    if(intro){
      fadeIn();
    }
    analogWrite(pinLed1,255);
  }
  else{
    if(outro){
      fadeOut();
    }
    analogWrite(pinLed1,0);
  }
}

void fadeIn(){
  for(value=0;value<=255;value++){
    analogWrite(pinLed1,value);
    delay(30);
    
  }
  intro=false;
  outro=true;
}

void fadeOut(){
  for(value=255;value>=0;value--){
    analogWrite(pinLed1,value);
    delay(10);
  }
  intro=true;
  outro=false;
}
