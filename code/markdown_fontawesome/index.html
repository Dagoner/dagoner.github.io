Use fontawesome in markdown

2020.04.26

是否合适

尽管在markdown中使用fontawesome来添加小图标不太符合makrdown专注于内容而非形式的宗旨，并且在一些情况下这些小图标格式会有所改。但是，万一用的到呢 : )
例如本站中的about页面，但本站使用hugo搭建，而直接在markdown中以html的形式使用fontawesome并不会在网页中渲染出来，但是将使用fontawesome的html语句写成shortcode并在markdown页面中使用则可行。

如何使用

首先一如既往的在markdown文件头部引入fontawesome库

<head> 
    <script defer src="https://use.fontawesome.com/releases/v5.0.13/js/all.js"></script> 
    <script defer src="https://use.fontawesome.com/releases/v5.0.13/js/v4-shims.js"></script> 
</head>

然后直接在markdown文档中使用

<i class="{{ index .Params 0 }}"></i> xxx
