Hugo-Change the color of the navigation bar TAB

2020.04.26

How it works

列出所有的section，并通过 if in 检查当前section是否在文章的当前链接中，例如本blog有三个section: code, interaction, about,当进入code section 并浏览本文时，当前文章的url为http://39.97.210.17/code/hugo_navColor/, 此时可检查code section这个字符是否在url中从而改变导航栏选项卡的颜色，具体效果可参考此blog

<div class="nav_tags">
        {{ $url := .URL}}
        {{ range .Site.Sections }}
      <a href="{{ .URL }}" style="{{if in $url .URL }} background-color:yellow; {{end}}">{{.Title}}</a>
        {{ end }}
        <a href="{{ .Site.BaseURL }}about" style= "{{ if in $url "about"}} background-color:yellow; {{end}}">About</a>&nbsp;&nbsp;
      </div>
